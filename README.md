## 🚀 **Quick Start Guide**

### **Prerequisites**

```bash
✅ TypeScript 4.5+
✅ Node Package Manager (npm)
✅ Modern development environment
```

### **Installation & Setup**

#### 1️⃣ **Clone Repository**

```bash
git clone https://github.com/vansh1233/Notify
cd Notify
```

#### 2️⃣ **Install Dependencies**

```bash
# Using npm
npm install

# Using yarn (alternative)
yarn install
```

#### 3️⃣ **Launch Development Server**

```bash
npm start
```

#### 4️⃣ **Run Test Suite**

```bash
npm test
```
